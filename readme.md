# DLN RTX Tile Refiner

An advanced tile-based refiner for ComfyUI that splits images into grids (2x2, 4x4, 8x8...), enhances each tile using **Nvidia RTX Super Resolution (nvvfx)**, and refines them with a KSampler pass.

## 🚀 Features
- **Refiner & Upscale Modes:**
    - **Refiner (Same Size):** Keeps the original resolution but uses RTX to generate micro-details and textures that are then "baked" back into the original size.
    - **Upscale (Larger Image):** Generates a high-resolution output (e.g., 4K) based on the RTX factor, preserving every new detail generated by the KSampler.
- **Nvidia RTX Integration:** Uses `nvvfx` (Nvidia Tensor cores) to upscale individual tiles before refining, providing a higher pixel density for the KSampler to work on.
- **Seamless Merging:** Implements a linear-blend overlap algorithm to eliminate visible seams between tiles (now with multi-resolution support).
- **Real-Time Logging:** Detailed console output showing the progress of each tile and integrated ComfyUI progress bar.
- **Smart Edge Handling:** Automatically crops tiles at the edges to prevent dimension mismatch errors.

## 🛠️ Inputs
- **image:** The source image to refine.
- **model / positive / negative / vae:** Standard ComfyUI sampling inputs.
- **seed / steps / cfg / sampler / scheduler:** Standard KSampler settings.
- **denoise:** Recommended between `0.15` and `0.35` for refinement.
- **grid_divisions:** Choose how many parts to divide the image into.
- **overlap:** Pixels of overlap for blending (default `64`).
- **rtx_upscale_factor:** Scale each tile before refining (requires Nvidia RTX GPU).
- **rtx_quality:** Quality level for Nvidia Super Resolution (LOW to ULTRA).

## 📦 Installation
1. Clone or move this directory to your `custom_nodes` folder.
2. Install dependencies:
   ```bash
   pip install nvvfx  # Only for RTX GPUs
   ```
